# Sveltekit + Taiwind CSS + Capacitor App

### Clone Project

`git clone "https://github.com/mattiabottes/stca.git"`\
`cd stca`\
`npm install`

### Install Capacitor

`npm install @capacitor/core @capacitor/cli`\
`npx cap init --web-dir=build`\
`npm install @capacitor/android`\
`npx cap add android`\
`npx cap open android`
